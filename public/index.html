A project represents a Spotify clone created by VueJS, Vuex and Spotify API.

Current state of a project.

What is used for building app?

  • VueJS Framework
  • Vuex (state management)
  • Vue Router (route management)
  • Font Awesome 5 (icons)
  • Spotify API with Javascript Wrapper
  • Simplebar
