What is it about?

A project represents a Spotify clone developed by VueJS, Vuex, Vue Router and JS Wrapper for Spotify API.

VueJS Spotify Player uses a Spotify API for authentication and for the access to user's playlists and information about the user.

In the application is used Vuex Store for maintaining a current state of application and Vue Router for maintaining different routes in the application.

If you are interested in code you can go to the project repository.
